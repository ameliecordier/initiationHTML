Anatomy of a webpage
====================

.. code-block:: html

   <!DOCTYPE html>
   <html>
     <head>
       <title>My first web page</title>
       <meta charset="UTF-8">
     </head>
     <body>
       This is the body of my document,
       where I write the content
       of my web page. 
     </body>
   </html>
